## 八、部署与国内访问

### 8.1 问题分析

Vercel 默认域名 `*.vercel.app` 在国内被墙，原因是 GFW 屏蔽了 Vercel 的全球 CDN IP 段。**仅靠自定义域名 + ICP 备案无法解决**，因为：

- ICP 备案是针对**国内服务器**的准入要求，Vercel 服务器在境外，备案无法改变网络层封锁
- 墙是 IP 级别的封锁，即使绑了备案域名，流量仍然经过被墙的 Vercel IP，照样打不开

### 8.2 解决方案对比

| 方案 | 国内访问 | 备案要求 | 费用 | 推荐度 |
|------|:---:|:---:|------|:---:|
| **GitHub Pages** | 部分可访问（慢） | 不需要 | 免费 | ⭐⭐ |
| **Cloudflare Pages** | 偶尔可访问 | 不需要 | 免费 | ⭐⭐ |
| **Vercel + 自定义域名** | 不稳定 | 不需要 | 域名费 | ⭐ |
| **腾讯云 COS + CDN** | 全国秒开 | 需要 | 按量（低） | ⭐⭐⭐⭐⭐ |
| **阿里云 OSS + CDN** | 全国秒开 | 需要 | 按量（低） | ⭐⭐⭐⭐⭐ |
| **腾讯云 Webify** | 全国秒开 | 需要 | 免费额度 | ⭐⭐⭐⭐ |
| **Zeabur / Railway** | 不稳定 | 不需要 | 免费额度 | ⭐⭐⭐ |

### 8.3 推荐方案：腾讯云 COS + CDN

> 纯静态站点，最省钱的国内部署方案

1. **注册腾讯云账号**，完成实名认证
2. **申请 ICP 备案**（个人备案约 7-15 个工作日）
3. **创建 COS 存储桶**，开启静态网站托管
4. **配置 CDN 加速**，绑定已备案域名
5. **GitHub Actions 自动部署**：push 代码 → 自动上传到 COS

月费用：纯静态站点通常 **< 5 元/月**（CDN 流量费）

### 8.4 免备案临时方案

如果暂时不想备案，可以使用：

- **GitHub Pages**：`https://wangv5918.github.io/private-recipes/`，国内部分运营商可访问，速度较慢
- **Vercel + 自定义域名**：偶尔能通，不稳定，不推荐作为主力方案

## 九、分类体系重构 ✅

### 9.1 现状问题

| 问题 | 说明 |
|------|------|
| 烘焙/饮品子分类冗余 | `category=烘焙` 且 `subcategory=烘焙`，一级=二级，无实际分层意义 |
| 甜品缺失 | 双皮奶、冰粉、杨枝甘露等被归入"饮品"，但它们是"吃的"不是"喝的" |
| 饮品子分类过粗 | 35 条饮品全部归入"饮品"，茶饮/咖啡/奶茶/鸡尾酒混在一起 |

### 9.3 分类原则

- **烘焙** vs **甜品**：烘焙 = 烤箱制作（蛋糕/饼干/面包），甜品 = 非烘焙甜食（冻/冰/糖水）
- **饮品** vs **甜品**：饮品 = 液体（喝的），甜品 = 半固体/固体（吃的）
- 二级分类按**食材/做法/场景**细分，每个子分类至少 3 条
